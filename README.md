## Agentic AI & Claude Code

I use Claude Code daily in production work, not as an experiment. At Stretto, I've designed agentic AI workflows for security auditing, .NET migrations, and feature refinement. The security auditing pipeline alone cut detected risks by 90% on CI/CD with zero new CVEs.

**[claude-tools](https://github.com/mteknight/claude-tools)** is where I publish the reusable pieces of that work:

- `adr-generate`: an agent that runs a structured discovery conversation and produces a complete Architecture Decision Record, MADR-format, ready to commit.
- `diagram-generator`: a skill that automatically picks the right diagram type, Mermaid or ASCII, and generates it when a diagram explains a concept better than prose.

Both are single-file tools, no dependencies, no setup overhead. Drop them into `.claude/agents` or `.claude/skills` and they work.
